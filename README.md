**大切な人の美点を発見・記録する個人向けアプリ**

---

## 📖 プロジェクト概要

美点ノートは、身近な人の良いところを発見し、記録するためのWebアプリケーションです。

### 🎯 コンセプト

ANA全社員43,000人が実践し、松山空港の従業員満足度を日本一にした「美点発見」メソッドを、個人がポケットに入れて持ち歩けるようにしました。

### ✨ Phase 1の特徴

- **完全ローカル動作**: サーバー不要、IndexedDBで安全に保存
- **3人まで登録**: 無料版の制限（Phase 2以降で拡張）
- **写真機能**: 正方形トリミング、自動圧縮
- **美点記録**: LINE風チャット形式で記録
- **PDF出力**: 全人物の美点を一括出力

## 🎓 使い方

### 1. 人物を追加

1. ホーム画面で **「新しい人を追加」** をクリック
2. 名前、関係性、出会った日を入力
3. 写真を選択（任意）→ トリミング
4. **「追加する」** をクリック

### 2. 美点を記録

1. 人物詳細画面で **「美点を追加」** をクリック
2. LINE風チャット画面で美点を入力（最大20文字）
3. ✏️ ボタンをクリックまたはEnterキーで送信

### 3. 美点を見返す

- 人物詳細画面で過去の美点を一覧表示
- 日付ごとにグループ表示

### 4. PDFで出力

1. ホーム画面で **「PDFで出力」** をクリック
2. 全人物の美点が1つのPDFにまとまります
3. 自動ダウンロード

## 🔒 セキュリティ・プライバシー

### データ保存方式

- **Phase 1**: IndexedDB（完全ローカル）
- データは**あなたのデバイスのみ**に保存
- サーバーへの送信は一切なし
- ブラウザのデータを消去すると削除されます

### 注意事項

⚠️ **Phase 1はバックアップ機能がありません**

- ブラウザのキャッシュクリアでデータが消えます
- 大切なデータは定期的にPDF出力してください
- Phase 2以降でクラウド同期に対応予定

---

## 🐛 トラブルシューティング

### データが表示されない

1. **ブラウザの対応確認**: Chrome/Safari/Firefox/Edgeを使用
2. **IndexedDBの確認**: 開発者ツール > Application > IndexedDB
3. **キャッシュクリア**: ページを再読み込み（Cmd+R / Ctrl+R）

### 写真がアップロードできない

1. **ファイル形式**: JPEG/PNG/GIF対応
2. **ファイルサイズ**: 5MB以下
3. **ブラウザの権限**: ファイル選択を許可

### PDFが生成できない

1. **人物の登録**: 最低1人以上の登録が必要
2. **ポップアップブロック**: ブラウザ設定を確認
3. **ダウンロード許可**: ブラウザのダウンロード設定を確認
